# 💼 Salary Prediction using Polynomial Regression

🚀 A Machine Learning project that predicts salary based on experience level using **Polynomial Regression**.  
This project also includes an interactive **Streamlit Web App** for real-time predictions.

---

## 📌 Project Overview

This project demonstrates how **Polynomial Regression** can model non-linear relationships between experience level and salary.

Unlike Linear Regression, Polynomial Regression captures complex patterns in data, providing more accurate predictions.

## ⚙️ How It Works

1. Load dataset  
2. Split into input (Level) and output (Salary)  
3. Train Linear Regression model  
4. Convert data into polynomial features  
5. Train Polynomial Regression model  
6. Predict salary based on user input  
7. Visualize results

## ▶️ Run Locally

```

cd salary-prediction-polynomial
```

### Step 2: Install Dependencies

```
pip install -r requirements.txt
```

### Step 3: Run Streamlit App

```
streamlit run app.py
```
